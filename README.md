## TONBUS Instruction 

 1. Download TONBUS project from github
 2. Start TON test client
    - ./test-lite-client -C config.json
 3. Create a new wallet
    - under test folder
    - ./fift -I <source dir>/lite-client/crypto/fift/lib new-wallet.fif
